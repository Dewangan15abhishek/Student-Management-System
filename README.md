# Student Management System (C++)

A console-based Student Management System developed in C++.  
The program allows users to manage student records using file handling.

## Features
- Add new student records
- Search student by admission number
- Modify existing records
- Delete records
- Display all records
- Delete all data

## Technologies Used
- C++
- File Handling (fstream)
- Object-Oriented Programming

## How to Run

Compile:

g++ main.cpp -o student

Run:

./student
